Пять шагов — и ваш форум для испанского готов к использованию

Создайте аккаунт на GitHub

Зарегистрируйтесь бесплатно на github.com

Откройте шаблон

Перейдите на feekool.github.io — готовый шаблон для испанского

Создайте свой репозиторий

Нажмите "Use this template" → "Create a new repository"
Назовите, например: mi-espanol или club-espanol

Включите GitHub Pages

Settings → Pages → Branch: main → Save
Через 1–2 минуты форум будет доступен: https://ваш-ник.github.io/mi-espanol

🇪🇸 Настройте под испанский

Отредактируйте файлы в репозитории:
• index.md — приветствие «¡Bienvenidos!»
• config.json — названия разделов на испанском
• Добавьте уровни, темы, материалы

✨ Pro tip: В шаблоне уже есть пример структуры: «Gramática», «Vocabulario», «Práctica», «Cultura». Просто активируйте нужные разделы и наполняйте контентом!
