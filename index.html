📦 Установка за 1 минуту

Эти шаги универсальны — после установки вы сможете настроить форум именно для обучения испанскому

  1. 1
    Зарегистрируйтесь на GitHub Создайте бесплатный аккаунт на github.com
  2. 2
    Откройте шаблон
    github.com/feekool/feekool.github.io
  3. 3
    Нажмите "Use this template" → "Create a new repository". Назовите его, например, «spanish-community» или «curso-espanol»
  4. 4
    Включите GitHub Pages: Settings → Pages → ветка main → Save. Через минуту форум онлайн!
  5. 5
    Настройте под испанский! 🇪🇸 Замените тексты, добавьте разделы для уровней, загрузите материалы — всё в файлах вашего репозитория
💡 Для преподавателей: После установки отредактируйте файл config.json или index.md, чтобы добавить: названия разделов на испанском, приветственное сообщение «¡Bienvenidos!», правила сообщества.

🎯 Как использовать для испанского языка

🎓

Онлайн-курсы испанского

Структурируйте уроки по неделям, добавляйте домашние задания, собирайте вопросы студентов в одном месте.

👁️ Демо: Курсы
💬

Разговорный клуб

Еженедельные темы для обсуждения: «Mi viaje a España», «Películas en español», «Recetas latinas».

👁️ Демо: Клуб
📖

Чтение и перевод

Публикуйте короткие тексты, новости, отрывки из книг. Студенты переводят, комментируют, обсуждают.

👁️ Демо: Чтение
🎧

Аудио-практика

Загружайте подкасты, песни, диалоги. Студенты слушают и пишут ответы, пересказы, мнения на испанском.

👁️ Демо: Аудио
👥

Группы по уровням

Создайте отдельные разделы для начинающих (A1–A2) и продолжающих (B1–C1), чтобы материалы были релевантны.

👁️ Демо: Уровни
🏆

Челленджи и игры

«7 дней без русского», «Фото-квест по Мадриду», «Составь диалог» — геймификация повышает вовлечённость!

👁️ Демо: Игры
